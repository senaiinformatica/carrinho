◆ setVelocidades()

void Carrinho::setVelocidades ( float vy,
float vx )

Define as velocidades base de deslocamento do carrinho.

Parâmetros
vyVelocidade de avanço (frente), em percentual de 0 a 100.
vxVelocidade lateral (esquerda/direita), em percentual de -100 a 100.

Este método ajusta as velocidades de referência utilizadas pelo seguidor de linha. A velocidade de avanço (vy) controla o deslocamento para frente, enquanto a velocidade lateral (vx) pode compensar desalinhamentos ou deslocamentos laterais. Ambos os valores são automaticamente limitados aos intervalos válidos.

Observação
Alterar essas velocidades afeta o comportamento de seguirLinhaStep() e dos modos automáticos de corrida.

◆ setVx()

void Carrinho::setVx ( float vx)

Define apenas a velocidade base lateral (eixo X).

Parâmetros
vxVelocidade lateral em percentual (-100 a 100).

Controla o deslocamento lateral constante aplicado ao carrinho durante o seguimento da linha. Pode ser usado para corrigir pequenos desvios ou testar movimentações diagonais.

Observação
Valores positivos deslocam o carrinho para a direita, negativos para a esquerda.
