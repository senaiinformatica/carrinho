Descrição detalhada

Implementação do controle do carrinho seguidor de linha com rodas omni.

Este módulo concentra:

  • Leitura do array de sensores via MCP23X17 e cálculo do erro de linha por LUT
  • Controle de 4 motores DC com DRV8833 e PWM pelo LEDC do ESP32
  • Controlador PID simples para gerar a velocidade angular "omega"
  • Máquina de estados: CALIBRACAO, ESPERANDO_LARGADA e CORRIDA
  • Console serial para ajuste de ganhos e parâmetros em tempo de execução

Convenções:

  • Percentuais de velocidade: -100 a 100
  • Duty PWM: 0 a 255
  • Tempo dt em segundos
  • omega limitado por omegaMax

Público alvo: alunos do ensino médio iniciando com robótica e C++.
